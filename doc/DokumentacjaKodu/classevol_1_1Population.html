Detailed Description

Population class. It's main class of evol library. It provides main loop of genetic algorithms and some standard implementation of it. It can be derived or extended using observers sytstem which allows to take some additional action without changing any part of standard code. Feel free to override virtual functions of this class, but do it at your own risk. Overriding registerObserver and notify* method is NOT recommended.
