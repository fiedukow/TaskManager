Detailed Description

FitnessFunction class should allow to compare value of FF for 2 Subjects. This class SHOULD be able to calculate and keep FF value to compare with. The most important part of this class is operator> method with compare two FF values.

Note:
This class is not only functor but also kinda container for FF value.
